## Conventions (recap)

- All endpoints under `/v1/`
- `Authorization: Bearer <Supabase JWT>` required on protected routes
- Success: `{"data": ...}`
- Error: `{"error": {"code", "message", "details?": ...}}`
- Mutating endpoints accept optional `Idempotency-Key`
